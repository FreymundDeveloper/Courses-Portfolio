# Courses-Portfolio

A generic Single Page Application (SPA) of a portfolio of courses, developed based on Gulp.

Some technologies used:

* Gulp(4.x);
* JavaScript;
* HTML;
* CSS;
* Node;
* SCSS/SASS.

## Running the app

```bash
# Installation
$ npm i -g gulp-cli
$ npm i

# Running
$ gulp
```
